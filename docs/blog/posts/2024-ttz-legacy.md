# Legacy ttZ measurement published in JHEP

![ttZ event display](https://atlas.web.cern.ch/Atlas/GROUPS/PHYSICS/PAPERS/TOPQ-2020-20/figaux_01.png){ .blog-hero }

Our comprehensive Run 2 measurement of $t\bar{t}Z$ production has been published in [JHEP](https://doi.org/10.1007/JHEP07(2024)163). This analysis represents the state-of-the-art in $t\bar{t}Z$ physics at the LHC.

<!-- more -->

Using the full 140 fb$^{-1}$ dataset collected by ATLAS during LHC Run 2 (2015–2018), we measured:

$$\sigma_{t\bar{t}Z} = 0.86 \pm 0.04\,(\text{stat.}) \pm 0.04\,(\text{syst.})\,\text{pb}$$

in agreement with the Standard Model prediction.

## Three results in one paper

This analysis is notable for combining three distinct physics results:

1. **Differential cross sections** — the most precise unfolded distributions of the $t\bar{t}Z$ system, at both particle and parton level.
2. **EFT interpretation** — limits on a large number of dimension-6 SMEFT operators involving the top quark, using the differential distributions.
3. **Spin correlations** — the first measurement of spin correlations in $t\bar{t}Z$, disfavouring the no-correlation hypothesis at 1.8σ.
